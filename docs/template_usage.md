# P站样式模板使用指南

## 概述

本项目提供了一个P站（Pornhub）样式的公众号文章HTML模板，可以通过MCP工具和资源来使用。

## 功能特性

1. **MCP资源**：AI可以读取模板内容
2. **MCP工具**：AI可以根据内容数据生成符合模板样式的HTML

## 使用方法

### 方式一：通过MCP资源读取模板

当AI需要查看模板结构时，可以访问资源：
- 资源URI: `template://phub_template`
- 资源名称: `P站样式模板`
- 描述: P站（Pornhub）样式的公众号文章HTML模板

### 方式二：通过MCP工具生成内容

使用 `wechat_template` 工具，根据提供的内容数据生成HTML。

#### 工具参数说明

**action**: 操作类型
- `generate`: 根据内容生成HTML
- `get_template`: 获取模板内容

**内容参数**:
- `title`: 文章标题（必需）
- `intro`: 文章介绍段落（可选）
- `image`: 顶部图片占位符文本（可选）
- `warning`: 警告提示文本（可选）
- `sections`: 文章章节列表（必需）
  - `title`: 章节标题
  - `content`: 章节正文内容
  - `stats`: 统计数据（可选）
    - `number`: 统计数字
    - `label`: 统计标签
  - `quote`: 引用文本（可选）
  - `features`: 功能列表（可选）
    - `title`: 功能标题
    - `description`: 功能描述
  - `code`: 代码块内容（可选）
  - `progress`: 进度条百分比（0-100，可选）
  - `image`: 章节图片占位符文本（可选）
- `tags`: 标签列表（可选）
- `actionButton`: 行动按钮（可选）
  - `text`: 按钮文本
  - `url`: 按钮链接
- `footer`: 页脚文本列表（可选）

## 使用示例

### 示例1：基本使用

当用户说"使用p站模板生成一篇关于AI技术的文章"时，AI可以：

1. 首先读取模板资源了解结构
2. 使用 `wechat_template` 工具的 `generate` 操作生成HTML

### 示例2：工具调用示例

```json
{
  "action": "generate",
  "title": "AI技术的最新发展",
  "intro": "在人工智能快速发展的今天，各种新技术层出不穷。",
  "sections": [
    {
      "title": "核心技术",
      "content": "AI技术的核心包括机器学习、深度学习等。",
      "features": [
        {
          "title": "机器学习",
          "description": "让计算机从数据中学习"
        },
        {
          "title": "深度学习",
          "description": "模拟人脑神经网络"
        }
      ]
    }
  ],
  "tags": ["AI", "机器学习", "深度学习"]
}
```

## 模板样式特点

- **黑色背景** (#000000)
- **橙色强调色** (#ff9900)
- **圆形数字图标** 用于章节标题
- **响应式设计** 适配移动端
- **多种内容块** 支持统计、引用、代码、进度条等

## 注意事项

1. 模板工具会自动转义HTML特殊字符，确保内容安全
2. 代码块内容不会被转义，可以包含HTML代码
3. 生成的HTML可以直接用于微信公众号的图文消息
